I have knowledge in HTML, CSS, preprocessors (Sass, Scss, Less), JavaScript, Bootstrap library, can apply Bootstrap-Grid to create adaptive websites. I can create Python scripts and GUI software (using PyQT, for example) in addition to Robot Operating System and CoppeliaSim. But I believe that the most important skill is to be able to search for information and apply it to the task at hand. Why reinvent the wheel? In 2020, I graduated with a bachelor's degree in Mechatronics and Robotics from the Russian Technical University (RTU MIREA). On the basis of my thesis about applying Neural Networks with image processing, I created a Telegram-bot for processing (deblurring using Neural network) and creating panoramic images. Unfortunately, this bot is currently offline. From 2019 to 2020 I worked as manual QA engineer in VK holding company's service testing program called "/testpool". Since 2021 I have been working as a QA and Automatization engineer at 1C International. We use an internal framework based on Cucumber as well as SikuliX and Cypress to test desktop application, mobile application, and web version of ERP product. To improve my skills, I am currently taking a Web development and website testing courses at Udemy.com, which includes an in-depth study of JavaScript incl. the React, Redux, and Cypress libraries. In the future, I want to learn React Native to develop cross-platform applications.
