Утилиты:

* [PassGen](https://github.com/kakoytochelik/kakoytochelik.github.io/tree/main/passgen) - простой генератор паролей на Python. Есть две опции: первая - быстрая генерация с сохранением в буфер обмена (без опций), вторая - с заполнением названия сервиса (может брать адрес из буфера обмена) и указанием длины пароля; сохраняет как в буфер обмена, так и в локальный текстовый файл.
* DeblurPano - часть выпускной работы, только без "железной" части. Алгоритм сшивания изображений (кадров видео) в панораму (OpenCV) с использованием генеративной состязательной сети для устранения размытий на кадрах (DeblurGAN). На основе этого проекта сделан телеграм-бот @DeblurPano_bot с отдельным функционалом как для построения панорам из готовых изображений и видео, так и для устранения размытия на отдельном изображении. Основная часть работает, надо только доработать сценарий использования сразу несколькими пользователями.
