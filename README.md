# Утилиты:

* [@PyPassGen_Bot](https://t.me/pypassgen_bot) - Telegram-бот, генерирующий пароли. Две опции - быстрая генерация, и генерация с указанием длины пароля, а также (опционально) названия сервиса, для которого пароль генерируется (для менеджмента паролей прямо в телеграме). Бот основан на следующей в списке утилите. Написан на Python с использованием библиотеки python-telegram-bot.
* [PassGen](https://github.com/kakoytochelik/kakoytochelik.github.io/tree/main/passgen) - простой генератор паролей на Python. Есть две опции: первая - быстрая генерация с сохранением в буфер обмена, вторая - с заполнением названия сервиса (может автоматически брать адрес из буфера обмена) и указанием длины пароля; сохраняет как в буфер обмена, так и в локальный текстовый файл. Использует цифры и буквы разного регистра. Создал "от нечего делать" для себя, так как иногда браузер не предлагает сгенерировать пароль, а существующие генераторы создают пароли "некрасивого" формата. В планах - перевести на JS и сделать UI под разные устройства. Пока доступен телеграм-бот.
* [@DeblurPano_bot](https://t.me/deblurpano_bot) - Telegram-бот, предназначенный для устранения размытия с изображений с целью дальнейшей их склейки в панораму. Может создавать панорамы как из отдельных изображений, так и из видео. Основан на выпускной работе, только без "железной" части. Алгоритм сшивания изображений (кадров видео) в панораму выполнен с помощью библиотеки OpenCV, устранение размытий - с использованием генеративной состязательной сети DeblurGAN. Бот написан с помощью python-telegram-bot. Функционал работает, однако надо работать над скоростью обработки и сценарием использования сразу несколькими пользователями. Работает не круглосуточно.
