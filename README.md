Утилиты:

* passgen - простой генератор паролей на Python. Есть две опции: первая - быстрая генерация с сохранением в буфер обмена (без опций), вторая - с заполнением названия сервиса (может брать адрес из буфера обмена) и указанием длины пароля; сохраняет как в буфер обмена, так и в локальный текстовый файл.
* DeblurPano - часть выпускной работы, только без "железной" части. На его основе сделан телеграм-бот @DeblurPano_bot
